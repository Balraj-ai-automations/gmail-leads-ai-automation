# Gmail Leads AI Automation

## Overview
AI-powered Gmail lead automation system that captures inbound emails, extracts lead information, qualifies opportunities using AI, and streamlines response workflows.

## Current Status
Phase 1 MVP

## Current Features
- Gmail email trigger
- Lead extraction workflow
- Basic automation logic
- n8n workflow implementation

## Planned Features
- AI lead qualification
- Email response generation
- Lead memory (Vector DB)
- CRM integration
- Follow-up automation
- Human approval workflow

## Tech Stack
- n8n
- Gmail API
- OpenAI API
- ChromaDB / Supabase
- Webhooks

docs/ → project documentation
